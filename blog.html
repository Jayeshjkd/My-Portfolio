My Portfolio Website Journey

Building My Personal Portfolio

As a BCA student, I wanted to create a personal portfolio website to showcase my skills, projects, and learning journey. Building this website helped me improve my knowledge of HTML and CSS while gaining practical experience in web development.

Website Structure and Design

The website includes sections such as Home, About Me, Skills, Projects, Services, and Contact. I used HTML to create the structure of the website and CSS to design a clean and responsive layout.

Responsive Design Challenges

During development, I faced several challenges, including image alignment, spacing issues, and responsive design. The most challenging part was creating a hamburger menu for mobile devices. After several attempts and testing, I successfully implemented it.

Building Confidence in Frontend Development

This project increased my confidence in frontend development and motivated me to continue learning JavaScript and modern web technologies. In the future, I plan to add more projects and new features.

Looking Back on This Project

Building this portfolio has been a valuable learning experience and an important step in my journey as a web developer.
